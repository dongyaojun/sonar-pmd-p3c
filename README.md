# SonarQube PMD-P3C Plugin 

## Preface
sonar-pmd是sonar官方的支持pmd的插件，但是还不支持p3c，需要在pmd插件源码中添加p3c支持
p3c是阿里在pmd基础上根据阿里巴巴开发手册实现了其中的49开发规则。
